## 3. FAILURE TAXONOMY (IMPORTANT)

For each failure, classify into:

* **REI (reasoning–execution mismatch)**
  Model knew the fix but didn’t implement it

* **Heuristic error**
  Model applied wrong pattern or shortcut

* **Causal error (CSF)**
  Model failed to understand system dynamics

* **Structural / parsing error**
  Code didn’t run, syntax error, etc.

* **Ambiguous / other**

Explain WHY you chose the category.
